# VoiceType

Нажал горячую клавишу — сказал — текст появился там, где стоит курсор. Работает в любом приложении на Windows.

![VoiceType](docs/preview.png)

Распознавание через Whisper API (OpenAI или OpenRouter). Записи сохраняются — можно переслушать, скопировать, вставить повторно.

## Установка

Скачать [VoiceType-Setup.exe](https://github.com/kirill-kopylov/voice-type/releases/latest) и запустить. При первом запуске указать API-ключ в настройках.

## Как пользоваться

1. Открой любое приложение и поставь курсор в текстовое поле
2. Нажми **Ctrl+Shift+H** — услышишь звуковой сигнал, в углу экрана появится индикатор с визуализацией голоса
3. Говори
4. Нажми **Ctrl+Shift+H** ещё раз — второй сигнал, индикатор переключится на анимацию обработки
5. Через пару секунд распознанный текст вставится в поле автоматически

Приложение работает в фоне из системного трея. Окно можно закрыть — запись по горячей клавише продолжит работать.

## Возможности

- **Запись из любого приложения** — глобальная горячая клавиша работает поверх всего
- **Автовставка** — текст сразу появляется там, где курсор. Работает в браузерах, мессенджерах, IDE, терминалах
- **История** — все транскрипции сохраняются. Можно искать по тексту, переслушивать аудио, копировать или вставлять повторно
- **Визуализация** — во время записи видно live-волну голоса, во время обработки — анимацию
- **Звуковые сигналы** — разные звуки на начало и конец записи, чтобы не смотреть на экран
- **Буфер обмена** — по умолчанию после вставки буфер обмена восстанавливается к тому, что было до записи
- **Автозапуск** — можно включить запуск вместе с Windows
- **Два провайдера** — OpenAI и OpenRouter, переключается в настройках
